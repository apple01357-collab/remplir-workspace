✦ AI 자동 분류 사용법
자유롭게 입력하면 AI가 자동 분류합니다
여러 항목을 한 번에 입력해도 됩니다. 카테고리를 명시하지 않아도 문맥으로 판단합니다.

예시: "내일까지 거래처 미팅 준비 우선, 3호기 이상 소음 이슈 진행중, A4용지 구매 필요, 마케팅 자동화 아이디어"

📎 사진 첨부
클립 버튼으로 사진을 첨부할 수 있습니다. 첨부된 사진은 항목에 함께 저장되어 대시보드에서 확인 가능합니다. 텍스트와 함께 입력하면 AI가 텍스트 내용을 분류합니다.
🔄 상태 변경
항목 카드의 상태 버튼으로 전환
업무: 우선 → 단기 → 장기 → 완료 순으로 전환 (완료 시 취소선 표시)
업무 → Issue Log: 업무 항목의 ⚠ Issue화 버튼 클릭 → Issue Log 진행중으로 이동
Issue Log: 진행중 → 해결완료 → TS작성완료 순으로 전환
필요 물품: 필요 → 구매예정 → 구매완료 순으로 전환
주기적 업무: 반복/주기 업무는 상태 전환 없이 고정
🔍 필터
상단 카테고리 카드를 클릭하거나, 필터 탭으로 원하는 항목만 볼 수 있습니다. 대화창에서 AI에게 직접 물어볼 수도 있습니다.
